\# Ambient Weather Hourly Data



This repository stores averaged weather data from the Ambient Weather Network.



\## Data Characteristics

\- Source: Ambient Weather Network API

\- Frequency: Every 30 minutes

\- Storage: JSON

\- Timezone: IST (Asia/Kolkata)

\- Records: 2 averaged data points per hour



\## Automation

Data is automatically fetched, averaged, and committed using GitHub Actions.
